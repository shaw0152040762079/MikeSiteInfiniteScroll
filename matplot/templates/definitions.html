Training data
  • Used to train model parameters Validation Data
  • used to determine what model hyperparameters to adjust test data
  • used to get some final performance Metric
RECALL
  • Ability of a model to find all relevant cases within a data set
  • number of true postives / number of true postives + false negatives
PRECISION
  • ability of a classficiation mdoel to identify only relevant data points
  • number of true positves / number of true positives + false positives
Precision VS recall
  • Often a trade off between recall and precision
F1 Score
  • In cases where we want to find an optimal blend of the two
  • Uses harmonic mean instead of a simple average to punish extreme values
  • A classifier with a precision of 1.0 and a recall of 0.0 with a simple average of 0.5 but an f1 of 0 f1 = 2* (precision * recall/ precision + recall)
Mean absolute error (MAE)
  • Mean of the the absolute value
  • Won't punish large errors
Mean absolute squared error (MSE)
  • Mean of the squared errors
  • Larger errors are more noted here compared to MAE
